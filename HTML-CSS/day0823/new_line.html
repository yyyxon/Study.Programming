HTML(Hyper Text Markup Language)로 Tim, Berners Lee가 고안한 웹에서 동작하는 언어입니다.
Hyper Link를 가지고 있어서 Web상의 다른 HTML과 연결하는 일을 하게 됩니다.

HTML을 Web Server로부터 요청하고 응답 받는 프로토콜은 HTTP로 Tim, Berners Lee가 고안하였습니다.
HTML5에 와서는 HTML은 구조를 JavaScript은 연산을 CSS는 디자인을 수행한다.

HTML은 연산과 제어의 기능이 없는 언어

Web Browser는 사용자가 정의한 tag를 그려서(Rendering) 보여주는 일, 에러가 발생하지 않는다.
